API Endpoints

GET /api/achievements

Retrieve a list of all achievements.

[
  {
    "achievementId": "a1b2c3d4-e5f6-7890-1234-567890abcdef",
    "name": "Task Master",
    "description": "Complete 100 tasks.",
    "criteria": {
      "type": "TASK_COMPLETION",
      "count": 100
    },
    "createdAt": "2024-07-28T10:00:00"
  }
]
GET /api/achievements/{achievementId}

Retrieve a specific achievement by its ID.

{
  "achievementId": "a1b2c3d4-e5f6-7890-1234-567890abcdef",
  "name": "Task Master",
  "description": "Complete 100 tasks.",
  "criteria": {
    "type": "TASK_COMPLETION",
    "count": 100
  },
  "createdAt": "2024-07-28T10:00:00"
}
POST /api/achievements

Create a new achievement.

{
  "name": "New Achievement",
  "description": "Description of the new achievement",
  "criteria": {
    "type": "POINTS_EARNED",
    "threshold": 5000
  }
}
PUT /api/achievements/{achievementId}

Update an existing achievement.

{
  "name": "Updated Achievement Name",
  "description": "Updated description of the achievement",
  "criteria": {
    "type": "POINTS_EARNED",
    "threshold": 10000
  }
}
DELETE /api/achievements/{achievementId}

Delete an achievement.

No content returned on success.
GET /api/achievements/user/{userId}

Retrieve a user's achievements.

{
  "userId": "user123",
  "achievements": [
    {
      "achievementId": "a1b2c3d4-e5f6-7890-1234-567890abcdef",
      "name": "Task Master",
      "description": "Complete 100 tasks."
    }
  ]
}
POST /api/achievements/process/{userId}

Process an event for a user (e.g., TASK_COMPLETED).

{
  "eventType": "TASK_COMPLETED",
  "eventDetails": {
    "taskId": "task-789",
    "priority": "HIGH"
  }
}
