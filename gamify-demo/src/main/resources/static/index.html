API Endpoints

GET /api/users

Retrieve paginated list of all registered users

GET/api/users/{id}

Retrieve a specific user by ID

POST/api/users

Create a new user

PUT/api/users/{id}

Update an existing user

DELETE/api/users/{id}

Delete a user

Rewards

GET/api/rewards

Retrieve a list of all rewards

GET/api/rewards/{id}

Retrieve a specific reward by ID

POST/api/rewards

Create a new reward

PUT/api/rewards/{id}

Update an existing reward

DELETE/api/rewards/{id}

Delete a reward

Redemptions

POST/api/redemptions

Redeem a reward for an employee

GET/api/employees/{id}/redemptions

Get all redemptions for a specific employee

Achievements

GET/api/achievements

Retrieve a list of all achievements

GET/api/achievements/{id}

Retrieve a specific achievement by ID

POST/api/achievements

Create a new achievement

PUT/api/achievements/{id}

Update an existing achievement

DELETE/api/achievements/{id}

Delete an achievement

User Achievements

GET/api/user_achievements/user/{userId}

Retrieve all user achievements for a specific user

GET/api/user_achievements/{id}

Retrieve a specific user achievement by ID

POST/api/user_achievements

Create a new user achievement (e.g., grant an achievement to a user)

Leaderboard

GET/api/leaderboard

Retrieve the leaderboard

About Gamify API

Gamify API is a comprehensive employee gamification platform designed to boost engagement, productivity, and satisfaction in the workplace. Our platform leverages the power of gamification to create a more dynamic and rewarding work environment.

Built with modern technologies and best practices, Gamify API offers a robust and scalable solution for businesses of all sizes.
